Recent Work:

For Progress Coffee, I did a complete website overhaul for an Austin-area wholesale coffee roasting company that I worked for part-time while I was in school. Challenges I ran into included figuring out how to route and update shopping cart information easily as a user, and how to use server requests to send an email via a contact form.


Technologies Used: Node server, Express routing, React component rendering, Snipcart shopping cart, Mailgun contact form, Materialize design, Parallax template.

Classify is a learning management system built to help teachers easily distribute course materials and resources to their students. My specific contributions focused on account creation -- making sure a user could create an account, that duplicate account names could not be made, and that session information was specific to the user that was signed in to use the site.


Technologies Used: Node server, Express routing, Express Session, brypt password encryption, MySQL Database, Sequelize DB ORM, Materialize design.


My Teammates: Lisa Battle, Chance Eakin, John Flores, and Derrick Suarez.
